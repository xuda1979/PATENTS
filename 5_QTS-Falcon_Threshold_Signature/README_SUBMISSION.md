# 专利申请文件准备说明

已为您生成符合中国国家知识产权局（CNIPA）格式要求的专利申请文件。所有文件均为 Word (.docx) 格式，位于当前目录下。

## 文件列表

1.  **100001_权利要求书.docx**
    - 包含所有权利要求。
    - 格式：宋体，小四号（12pt），1.5倍行距。
    - 请检查编号是否连续。

2.  **100002_说明书.docx**
    - 包含发明名称、技术领域、背景技术、发明内容、附图说明、具体实施方式。
    - 格式：宋体，小四号（12pt），1.5倍行距。

3.  **100003_说明书摘要.docx**
    - 包含摘要文本。
    - 格式：宋体，小四号（12pt），1.5倍行距。

4.  **100004_说明书附图.docx**
    - **注意**：此文件目前仅包含图片说明和占位符。
    - **操作**：请打开此文件，将实际的图片（黑白线条图）插入到对应的“[在此处插入图片]”位置。
    - 图片要求：清晰的黑白线条图，无灰度，分辨率至少 300 DPI。

5.  **100005_摘要附图.docx**
    - **注意**：此文件用于存放最具代表性的一幅附图（通常是图1）。
    - **操作**：请在此文件中插入图1。

6.  **100006_专利请求书.docx**
    - 包含申请人、发明人等信息。
    - 此文件通常用于参考，正式提交时需在电子申请客户端（CPC）或网页版系统中填写表格。

## 提交前检查清单

- [ ] 打开所有 Word 文档，检查排版是否整齐。
- [ ] **重点**：在 `100004_说明书附图.docx` 和 `100005_摘要附图.docx` 中插入真实的图片。
- [ ] 检查 `100001_权利要求书.docx` 中的引用关系（如“根据权利要求1所述...”）是否正确。
- [ ] 确认 `100006_专利请求书.docx` 中的申请人信息（地址、邮编、证件号）准确无误。

## 格式说明

- **字体**：宋体 (SimSun)
- **字号**：小四 (12pt)
- **行距**：1.5 倍
- **页边距**：上 2.5cm, 下 1.5cm, 左 2.5cm, 右 1.5cm
- **纸张**：A4

祝您的专利申请顺利！
