# 专利申请文件准备说明 (更新版)

已为您重新生成符合中国国家知识产权局（CNIPA）格式要求的专利申请文件。所有文件均为 Word (.docx) 格式，位于当前目录下。

## 文件列表

1.  **100001_权利要求书.docx**
    - **更新**：已修复权利要求编号问题，现在每项权利要求都以“1. ”、“2. ”等开头。
    - 格式：宋体，小四号（12pt），1.5倍行距。

2.  **100002_说明书.docx**
    - **更新**：已去除重复的“发明名称”标题，排版更整洁。
    - 包含技术领域、背景技术、发明内容、附图说明、具体实施方式。

3.  **100003_说明书摘要.docx**
    - 包含摘要文本。

4.  **100004_说明书附图.docx**
    - **注意**：此文件目前仅包含图片说明和占位符。
    - **操作**：请打开此文件，将实际的图片（黑白线条图）插入到对应的“[在此处插入图片]”位置。

5.  **100005_摘要附图.docx**
    - **注意**：此文件用于存放最具代表性的一幅附图（通常是图1）。
    - **操作**：请在此文件中插入图1。

6.  **100006_专利请求书.docx**
    - 包含申请人、发明人等信息。

## 提交前最终检查

- [ ] **权利要求书**：确认第一项权利要求以“1. ”开头，后续权利要求编号连续。
- [ ] **说明书**：确认第一页顶部居中为发明名称，随后是“技术领域”等标题。
- [ ] **附图**：务必插入真实的黑白线条图。
- [ ] **页码**：Word 文档通常会自动处理页码，但建议检查打印预览确保页码位于页面底部居中。

祝您的专利申请顺利！
