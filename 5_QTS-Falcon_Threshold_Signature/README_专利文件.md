# 专利申请文件说明

## 主要提交文件

本项目包含以下两个核心专利申请文件（Word 格式）：

### 1. 专利技术交底书.docx
- **内容**: 完整的技术交底文档，包含：
  - 发明名称
  - 申请人和发明人信息
  - 技术领域
  - 背景技术（现有技术问题分析）
  - 发明内容（六项核心创新点）
  - 附图说明
  - 具体实施方式
  - 技术效果
  - 权利要求书
  - 说明书摘要

### 2. 现有技术检索报告.docx
- **内容**: 完整的现有技术检索报告，包含：
  - 检索参数和关键词
  - A类：基础性现有技术
  - B类：相关门限签名现有技术
  - C类：MPC 协议现有技术
  - D类：跨链与区块链现有技术
  - E类：专利检索结果（USPTO、EPO、CNIPA）
  - 新颖性分析
  - 可专利性评估
  - 权利要求差异化矩阵
  - 建议申请策略

## 文件位置

```

├── 专利技术交底书.docx          ← 技术交底书 (主要文件1)

├── latex/

│   ├── compile_to_word_v2.ps1  ← 编译脚本
│   └── output/                 ← 编译输出目录
└── ...
```

## 如何重新生成

如果需要修改内容并重新生成 Word 文件：

1. 编辑 LaTeX 源文件（`latex/交底书.tex` 或 `latex/检索报告.tex`）
2. 运行编译脚本：
   ```powershell
   cd latex
   .\compile_to_word_v2.ps1
   ```
3. 生成的文件将保存在 `latex/output/` 目录

## 发明概述

**发明名称**: 面向量子计算网络的抗量子攻击节点协同认证系统及方法

**核心创新点**:
1. 基于 NTRU 结构的算术共享 NTT 协议
2. 具有正确参数校准的分布式高斯采样 (σ/√N)
3. 基于 Beaver 三元组预处理的协同拒绝采样（6轮常数通信）
4. 具有作弊检测的可验证秘密共享
5. 动态节点准入与密钥重构
6. 容错与超时处理

**技术效果**:
- 签名长度: ~666 字节（比 Dilithium 缩小 3.6 倍）
- 通信轮数: 6 轮常数（与节点数无关）
- Gas 费用: ~50,000 Gas（节省 72%）

## 注意事项

- 打开 Word 文件后，可能需要手动调整部分表格和公式格式
- 数学公式在 Word 中可能显示为文本格式，需要转换为公式对象
- 建议使用 Microsoft Word 2016 或更高版本打开
